## Описание игры

Игра реализована на Python с использованием библиотеки `numpy`.  
Компьютер случайным образом выбирает число и повторяет попытки, пока не угадает загаданное число.

## Функции

### random_predict(number: int) -> int
- **Описание:** Случайным образом угадывает число `number`.  
- **Параметры:** 
  - `number` — загаданное число (по умолчанию 1).  
- **Возвращает:** количество попыток, потребовавшихся для угадывания.

### score_game(random_predict) -> int
- **Описание:** Определяет среднее количество попыток, за которое алгоритм угадывает число на 1000 случайных примерах.  
- **Параметры:** 
  - `random_predict` — функция угадывания.  
- **Возвращает:** среднее количество попыток.
